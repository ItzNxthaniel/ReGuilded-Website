# ReGuilded Website

**ReGuilded** is a client injector/client mod that allows you to extend the functionality of Guilded's client by providing theme and addon support.

## Technologies
The technologies used to make the ReGuilded website:
- TypeScript
- Next.js
- React
- Chakra
